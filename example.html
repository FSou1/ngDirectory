Example data

Model objects

For example we have objects that references to directory, that contains displaying value by id: 

$scope.vehicle = {
    brandId: 1
};

$scope.discount = {
    brandIds: [2, 4, 5]
};

Directories

$scope.brands = [
    { id: 1, title: 'Audi' },
    { id: 2, title: 'BMW' },
    { id: 3, title: 'Cadillac' },
    { id: 4, title: 'Chery' },
    { id: 5, title: 'Chevrolet' }
];

Usage

Display single value

<pre ng-directory key="vehicle.brandId" source="brands"></pre> 




                        
Display multiple value

                        

                            
                                <pre ng-directory key="discount.brandIds" source="brands"></pre>
                            
                        

                        


                        
Custom display property name

                        

                            
                                <pre ng-directory key="discount.brandIds" source="brands" options="{ displayProperty: 'manufacturer' }"></pre>
